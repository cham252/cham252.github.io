Lab Architecture

Two fully isolated VirtualBox networks: a Corporate network (DC01, W11-Client, Ubuntu, Security Onion, Postfix, Wazuh) and an Attack network (Kali Linux). No traffic leaks between segments — simulates real air-gapped enterprise/attack-side segmentation.

Domain Controller
DC01
Windows Server 2022
Active Directory, Group Policy Objects, DNS. Primary domain controller — the final compromise target. Hosts LDAP, Kerberos, SMB. GPOs enforce password policy, audit logging, and baseline security controls.
Workstation Endpoint
W11-Client
Windows 11
Domain-joined corporate workstation. Primary initial access target — phishing link delivered here. Wazuh agent installed for endpoint telemetry and alert generation.
Linux Endpoint
Ubuntu Desktop
Ubuntu 22.04
Corporate Linux workstation on the domain network. Secondary endpoint target. Provides cross-platform telemetry and tests whether SIEM detects Linux-side lateral movement.
Network Detection · IDS/NSM
Security Onion
Zeek · Suricata · Elastic Stack
Full network security monitoring stack. Zeek generates connection logs and protocol analysis. Suricata fires IDS alerts on known attack signatures. Elastic Stack indexes, searches, and visualizes all events.
Mail Server
Postfix
Ubuntu Server · Postfix MTA
Internal SMTP mail server. Used to deliver phishing emails to W11-Client, simulating realistic initial access via spearphishing link. Allows testing of email-based detection and mail flow visibility.
SIEM · EDR Correlation
Wazuh
Wazuh Manager + Agents
Open-source SIEM and XDR platform. Agents installed on W11-Client and Ubuntu. Collects Windows Event Logs, Sysmon events, file integrity monitoring, and process events. Primary alert source during the attack chain.
Red Team · Attack Box
Kali Linux
Kali Linux 2024
Isolated on the Attack network. Hosts the phishing infrastructure, C2 framework, and post-exploitation tooling. Executes the full attack chain against the Corporate network targets from an isolated network segment.
// 02

Red Team Attack Chain

Initial Access
1
Phishing Email — Spearphishing Link
Phishing email crafted on Kali and delivered via the Postfix mail server to W11-Client. Email contains a malicious link designed to harvest credentials or deliver a payload. MITRE T1566.002 — Spearphishing Link.
Execution
2
Credential Harvesting on W11-Client
User on W11-Client interacts with the phishing link. Credentials captured via credential harvester (phishing site or payload execution). Wazuh agent on W11-Client fires process creation and network connection alerts. MITRE T1056 — Input Capture.
Persistence
3
C2 Callback — Establish Foothold
Compromised W11-Client establishes C2 callback to Kali Linux on the Attack network. Reverse shell or C2 beacon initiated. Security Onion Suricata fires network-based IDS alert. Zeek logs the anomalous outbound connection. MITRE T1071 — Application Layer Protocol C2.
Discovery
4
AD Enumeration — Domain Recon
From the established foothold on W11-Client, attacker enumerates Active Directory: domain users, groups, SPNs, and domain controller location. LDAP queries and tools like BloodHound or net commands used. Wazuh fires on AD enumeration activity. MITRE T1069, T1087.
Lateral Movement
5
Credential Abuse — Lateral Movement to DC01
Using harvested credentials or Kerberos ticket abuse (Pass-the-Hash / Pass-the-Ticket), attacker pivots from W11-Client toward DC01. SMB or WinRM used for remote code execution. Security Onion detects lateral movement network patterns. MITRE T1550.002.
Impact
6
Domain Controller Compromise
Attacker achieves SYSTEM or Domain Admin access on DC01 — full domain compromise. DCSync attack or Admin share access confirmed. Both Wazuh and Security Onion generate high-severity alerts. Full attack chain documented with detection timestamps, alert correlation, and blue team response playbook.
