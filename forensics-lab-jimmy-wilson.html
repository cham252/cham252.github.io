Examination Procedure

01
Environment Setup
Logged into the ProxMox virtual infrastructure at the UofL Digital Transformation Center lab environment and booted the Windows 11 VM. Confirmed VM integrity and access credentials. Autopsy 4.22.1 was pre-installed on the desktop.
02
Evidence Intake and Documentation
Located 2020JimmyWilson.E01 in Documents → Forensics. Documented file name, full path, and file size prior to ingest. Established chain of custody. Timezone confirmed as America/New_York (EST).
03
Case Creation and Image Ingest
Created new Autopsy case (name: JimmyWilson, examiner: Christopher Ham). Added 2020JimmyWilson.E01 as primary data source with all ingest modules enabled: Recent Activity, Hash Lookup, File Type Identification, Extension Mismatch Detector, Embedded File Extractor, Picture Analyzer, Keyword Search, Email Parser, Encryption Detection, Interesting Files Identifier, Central Repository, PhotoRec Carver, Virtual Machine Extractor, YARA Analyzer, iOS Analyzer (iLEAPP). Both data sources (E01 and extracted SYSTEM.vhd) completed ingest successfully.
04
Systematic Artifact Examination
Worked through six artifact categories: Images/Videos, Documents (HTML and Office), Email Messages, Web History, Web Search, and OS Accounts. Items of forensic interest bookmarked with tag “Jimmy Wilson.” Metadata captured per item: file name, full path, created time, file size, hash values.
05
Report Generation and Submission
Generated Autopsy HTML report from all tagged items. Report navigation: E-Mail Messages (2), Tagged Files (5), Tagged Images (1), Tagged Results (19), Web History (7), Web Search (10). Formal examiner report written using organizational template and submitted to HR.

// 04

Conclusions

1
Pattern of criminal planning activity
The combination of searches for handguns, identity theft methods, fast getaway cars, and how to disappear without a trace — conducted across two browsers on two consecutive dates — reflects deliberate, sustained research activity inconsistent with legitimate real estate employment duties.
2
Personal email use on corporate device
Jimmy Wilson used a personal Gmail account (wilsonjimmy807@gmail.com) to communicate with an external contact (robert.ripoff@gmx.com) on the corporate workstation. Subject line “Re: New email address” suggests coordination with this individual.
3
Named contacts of concern
Desktop files “Jose Badguy” and “Robert Ripoff” suggest the user maintained records associated with specific individuals. “Robert Ripoff” directly matches the email contact identified in section 03.3.
4
File concealment activity
Six files flagged for extension mismatch and four for suspected encryption, indicating possible deliberate concealment of content on the device.
5
External storage use
Nine USB devices attached to the workstation, indicating potential data exfiltration or external file transfers. Contents of these transfers are not determinable from the current image alone and warrant further investigation.
