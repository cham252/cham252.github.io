Red Team Attack Chain

Initial Access · MITRE T1566.002
1. Phishing Email via Postfix
Phishing email crafted on Kali and delivered via Postfix to W11-Client. Email contains a malicious link for credential harvest or payload delivery. Simulates spearphishing link initial access.
Execution · MITRE T1056
2. Credential Harvesting on W11-Client
User interacts with phishing link. Credentials captured via harvester. Wazuh agent fires process creation and network connection alerts. MDE-equivalent telemetry captured for analysis.
Persistence · MITRE T1071
3. C2 Callback — Establish Foothold
W11-Client establishes C2 callback to Kali on the Attack network. Reverse shell or C2 beacon initiated. Security Onion Suricata fires network IDS alert. Zeek logs anomalous outbound connection.
Discovery · MITRE T1069, T1087
4. AD Enumeration
From foothold on W11-Client, attacker enumerates AD: domain users, groups, SPNs, DC location. BloodHound or net commands used. Wazuh fires on AD enumeration activity.
Lateral Movement · MITRE T1550.002
5. Lateral Movement to DC01
Using harvested credentials or Kerberos ticket abuse (Pass-the-Hash / Pass-the-Ticket), attacker pivots toward DC01. SMB or WinRM for remote code execution. Security Onion detects lateral movement patterns.
Impact · Full Compromise
6. Domain Controller Compromise
SYSTEM or Domain Admin access achieved on DC01. DCSync or Admin share access confirmed. Both Wazuh and Security Onion generate high-severity alerts. Full attack chain documented with detection timestamps and blue team response playbook.
